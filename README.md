Update Modul
  ```
pkg update -y
 ```
 Upgrade Modul
 ```
pkg upgrade -y
````
Install Modul Git
```
pkg install git -y
```
Clone URL Git
```
git clone https://github.com/MasFha/TrackingDevz.git
```
Install Pip
```
pkg install python-pip -y
```
Ubah Direktorinya
```
cd  TrackingDevz
```
Install Requirements
```
pip3 install -r requirements.txt
```
Jalankan Filenya
````
python3 TrackingDevz.py
